# Route Rehearsal

Tiny starter project for a calm "dry run" route preview app.

## Version 1 goal

Build a very small web app that helps someone rehearse a drive to a location.

## First features

- Enter a start location
- Enter a destination
- Show a simple route preview
- Highlight a few important moments like turns or arrival

## Files

- `index.html` - page structure
- `styles.css` - basic styling
- `app.js` - starter interactions

## Next step
